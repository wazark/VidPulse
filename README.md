# 🎬 VidPulse

VidPulse é uma aplicação desktop para Windows desenvolvida em Python com PySide6, que permite baixar vídeos e áudios do YouTube de forma simples e rápida.

## 📌 Objetivo

Este projeto foi criado com fins educativos, para ajudar alunos a compreender:

* Estrutura de projetos Python profissionais
* Desenvolvimento de aplicações desktop (Qt / PySide6)
* Integração com bibliotecas externas (yt-dlp)
* Organização modular de código

## ⚙️ Tecnologias utilizadas

* Python 3.x
* PySide6 (Interface gráfica)
* yt-dlp (Download de vídeos)
* FFmpeg (Conversão de áudio)

## 📂 Estrutura do projeto

```
VidPulse/
│
├── main.py
├── requirements.txt
│
├── core/
│   ├── downloader.py
│   └── utils.py
│
└─── ui/
    └── main_window.py
```

## 🚀 Como executar

1. Clonar o repositório:

```
git clone https://github.com/SEU_USER/VidPulse.git
```

2. Instalar dependências:

```
pip install -r requirements.txt
```

3. Executar:

```
python main.py
```

## ⚠️ Requisitos adicionais

É necessário ter o FFmpeg instalado no sistema:
https://ffmpeg.org/download.html
